## TimeSpark

TimeSpark is a Kotlin-based Andriod appplicatin that help the user stay socially connectedby providing personalized "spark" suggestions based on the time of day they input
The app prompt the user to perform simple social actions (e.g sending a good morning message or calling a friend)
The progect also implements GitHUB version controlActions to ensure a robust and maintainable codebase

## Features

Allows the user to enter time of day
-Generates social interrection suggestion instantly
-Provides feedback for invaild input
-Includes a reset button to clear input and results
-Simple, clean and user-friendly interface

## How to use

 1.Open the application
 2.Enter a time of day (e.g Morning, Mid-Morning, Afternoon, Mid-Afternoon, Dinner, After-Dinner)
 3.Tap "suggest" button
 4.View the suggested interection
 5.Tap "Reset" to clear input and try again

 NOTE BE: The application currently accepts only predefined time categories as input as app logic

>Morning-Send a Good-Morning text to family members
>Mid-Morning-Reach out to a colleague with a quick "Thank You"
>Afternoon-Share a funny meme or an interesting link with a friend
>Mid-Afternoon-Send a quick thinking of you message
>Dinner-Call a friend or a relative for a 5-minute catch-up
>After-Dinner-Leave a thoughtful comment on a friend post
>Invaild input-Display "invaild input"
